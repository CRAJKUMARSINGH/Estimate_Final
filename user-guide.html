Quick Start Guide

1

Import Your First Estimate

Click the "Import Excel" button in the top-right corner to upload your existing Excel estimate files.

Supported Formats:

  • • .xlsx - Excel Workbook
  • • .xls - Excel 97-2003
  • • .xlsm - Excel Macro-Enabled
2

Navigate Dashboard

Use the navigation menu to switch between different views: Dashboard, Estimates, SSR Database, and Reports.

3

Create or Edit Estimates

Click on any estimate to view and edit its details, or create a new estimate from scratch.

Add Items

Click "Add Item" to add measurement and cost items

Edit Values

Click on any field to edit quantities, rates, or descriptions

4

Export Your Work

Export your estimates in multiple formats for sharing, printing, or further analysis.

Best Practices

Excel Import Tips

Naming Conventions

  • • Use "General Abstract" for main summary sheet
  • • Use "Measurement [Part Name]" for measurement sheets
  • • Use "Abstract of Cost [Part Name]" for abstract sheets

Data Structure

  • • Include headers in first row
  • • Use consistent column ordering
  • • Avoid merged cells in data areas

Calculation Accuracy

Rate Management

  • • Update SSR rates regularly
  • • Use market rates for current projects
  • • Include wastage factors where applicable

Quantity Verification

  • • Double-check measurement calculations
  • • Use standard conversion factors
  • • Validate totals before finalizing

Troubleshooting

Import Issues

Problem: Excel file won't import or shows errors

Solution: Check file format, ensure it's not password protected, and verify sheet naming conventions

Calculation Errors

Problem: Totals not updating or showing incorrect values

Solution: Check for empty cells, verify rate entries, and ensure quantities are numeric

Export Problems

Problem: Export fails or generates corrupted files

Solution: Clear browser cache, try different format, or check file size limits
