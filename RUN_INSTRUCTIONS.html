Run Instructions

Welcome to the Construction Estimation System! This document provides detailed instructions on how to run the application.

Step 1: Check Python Installation

Open Command Prompt or Terminal and run:

python --version

You should see output like: Python 3.8.10

If Python is not installed, download it from python.org.

Step 2: Install Required Packages

Navigate to the project directory and install dependencies:

pip install -r requirements.txt

If you encounter permission issues, try:

python -m pip install --user streamlit pandas numpy openpyxl

Step 3: Run the Application

You can run the application using one of these methods:

Method 1: Using Batch Files (Windows)

Double-click on one of these files:

  • run_app.bat - Full setup with dependency installation
  • quick_start.bat - Quick launch if dependencies are already installed
  • run_streamlit.bat - Direct launch of the Streamlit app

Method 2: Manual Launch

Open Command Prompt, navigate to the project directory, and run:

streamlit run streamlit_estimation_app.py

Method 3: Using Python Module

Alternatively, you can run:

python -m streamlit run streamlit_estimation_app.py

Step 4: Access the Application

Once the application is running:

  1. The app will automatically open in your default web browser
  2. If not, manually navigate to: http://localhost:8501
  3. You should see the Construction Estimation System interface

üí° Pro Tips

  • To specify a custom port: streamlit run streamlit_estimation_app.py --server.port 8502
  • To stop the application, press Ctrl+C in the command prompt
  • If you encounter issues, check the install_log.txt file for error details

Troubleshooting Common Issues

"Module not found" errors

pip install --upgrade pip
pip install -r requirements.txt

Permission errors

Run Command Prompt as Administrator, or use:

python -m pip install --user -r requirements.txt

Port already in use

streamlit run streamlit_estimation_app.py --server.port 8502

Application doesn't open in browser

Manually navigate to http://localhost:8501 in your browser

Application Features

The Construction Estimation System includes:

  • Dashboard - Overview of project metrics and recent activity
  • Measurement Sheets - Create detailed measurement sheets for different work types
  • SSR Database - Manage Standard Schedule of Rates with search and filtering
  • Abstract of Cost - Generate cost abstracts with automatic calculations
